Weather Forecasting Using Machine Learning

This project predicts future temperature based on historical weather data using Machine Learning.

Tech Stack
• Python
• Pandas
• NumPy
• Scikit-Learn
• Matplotlib

Features
• Data preprocessing and cleaning
• Train/Test split for model validation
• Linear Regression model for prediction
• Visualization of temperature trends

Dataset
The dataset contains the following fields:
• Date
• Minimum Temperature
• Maximum Temperature
• Humidity

Working Process
1. Dataset import and cleaning
2. Feature selection
3. Model training
4. Forecasting and visualization

Output
The model predicts future temperature values and displays a graph comparing actual vs predicted results.
